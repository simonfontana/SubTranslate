SubTranslate

Translate video subtitles instantly while you watch. Click any subtitle word to pause and see its translation.

Chrome Web Store Firefox Add-ons

What is SubTranslate?

SubTranslate is a lightweight browser extension that lets you translate words and sentences directly from video subtitles. It works on YouTube, SVT Play, and svt.se, powered by the DeepL API.

How it works

  1. Play a video with subtitles enabled.
  2. Click a word in the subtitles — the video pauses and a translation tooltip appears.
  3. Double-click to translate the full sentence.
  4. Triple-click to translate all visible subtitles at once.
  5. Click any word in the translated sentence for a reverse translation back to the source language.
  6. Right-click subtitles to copy the text or look up a word on Wiktionary.

Features

  • Click individual subtitle words for instant translation
  • Double-click to translate the full sentence
  • Triple-click to translate all visible subtitles at once
  • Explore sentence translations with clickable words
  • Click translated words to see reverse translations back to the source language
  • Right-click subtitles to copy the text or look up a word on Wiktionary
  • Correctly handles hyphenated words split across subtitle lines
  • Automatically pauses playback during translation (configurable)
  • Tooltip closes when playback resumes
  • Works on YouTube, SVT Play, and svt.se
  • Supports all languages available through DeepL

Appearance & customization

  • Adjustable subtitle font size and vertical position
  • Customizable highlight color with presets and HSL color picker
  • Per-site settings overrides — configure each site independently

Advanced options

  • Context history — include previous subtitles for more accurate translations
  • Choice of DeepL translation model (high quality or fast)

Setup

  1. Install the extension from the Chrome Web Store or Firefox Add-ons.
  2. Open the extension popup and enter your DeepL API key.
  3. Choose your source and target languages, then save.

Open source
