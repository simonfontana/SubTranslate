What is SubTranslate?

SubTranslate is a lightweight browser extension that lets you translate words and sentences directly from video subtitles. It works on YouTube and SVT Play, powered by the DeepL API.

How it works

  1. Play a video with subtitles enabled.
  2. Click a word in the subtitles — the video pauses and a translation tooltip appears.
  3. Double-click to translate the full sentence.
  4. Click any word in the translated sentence for a reverse translation back to the source language.

Features

  • Single-click word translation
  • Double-click full sentence translation
  • Reverse translation (click translated words)
  • Supports 29 languages via DeepL
  • Works on YouTube and SVT Play

Setup

  1. Install the extension from the Chrome Web Store or Firefox Add-ons.
  2. Click the SubTranslate icon in your browser toolbar.
  3. Enter your DeepL API key (free tier works fine).
  4. Choose your source and target languages, then save.

Open source

SubTranslate is open source and contributions are welcome. Check out the GitHub repository.
